###  Öne Çıkan Projeler

####  Backend Mimarisi: Enterprise Game Backend System
* **Açıklama:** Oyun meta verilerini (yayıncı, yapımcı, tür vb.) merkezi olarak yöneten, yüksek trafiğe uygun ölçeklenebilir backend servisi. 
* **Teknik:** Java, Spring Boot, PostgreSQL, RESTful API. Veri kapsülleme için DTO (Data Transfer Object) deseni, performans için Indexing yapıları ve One-to-Many ilişkisel veritabanı tasarımı kullanıldı.

####  Siber Güvenlik: Network Anomaly Detection System (ARP/MAC Spoofing)
* **Açıklama:** Yerel ağ trafiğini analiz ederek "Man-in-the-Middle" ve "ARP Spoofing" saldırılarını gerçek zamanlı tespit eden güvenlik yazılımı.
* **Teknik:** Java ile Pcap4J üzerinden düşük seviyeli ham paket analizi (Packet Sniffing) ve IP-MAC eşleşmelerini dinamik HashMap ile kontrol eden anormallik algoritması.

####  Simülasyon: 3D FPS Engine Mimarisi
* **Açıklama:** Unity ve C# kullanılarak geliştirilen, modüler silah sistemleri ve gelişmiş yapay zeka davranışları içeren simülasyon.
* **Teknik:** Kalıtım, Polimorfizm ve Interface yapılarıyla tam modüler OOP mimarisi.

####  Makine Öğrenmesi: IDS Project (NSL-KDD)
* **Açıklama:** Siber saldırı tespiti için AdaBoost ve Random Forest algoritmaları kullanılarak geliştirilen sınıflandırma modeli.
* **Teknik:** Veri ön işleme, özellik seçimi ve GridSearchCV ile model optimizasyonu.
